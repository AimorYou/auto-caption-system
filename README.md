# auto-caption-system

 ## Windows (Python 3.9)
    1. git clone https://github.com/shashankx86/dlib_compiled.git
    2. pip install dlib_compiled/dlib-19.22.99-cp39-cp39-win_amd64.whl
    3. pip install -r requirements.txt
